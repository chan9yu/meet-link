## 🚀 Getting Started

로컬환경에서 앱을 실행하는 방법입니다.

### Prerequisites

해당 앱을 실행시키기 위한 필수 조건 입니다.

1. pnpm install

```shell
# pnpm 설치
npm -g install pnpm
```

2. node setup

```shell
# .nvmrc 기준 node version 설치
# nvm을 사용하지 않는다면 수동으로 노드버전을 맞춰주세요.
nvm use
```

### Installation

1. 프로젝트 클론

```shell
git clone https://github.com/chan9yu/meet-link
```

2. 프로젝트 디렉토리로 이동

```shell
cd meet-link
```

3. 종속성 설치

```shell
pnpm install
```

4. 개발 서버 시작

```
pnpm dev
```
